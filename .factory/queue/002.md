# Add neo-brutalist card styles to PostCard.vue

## Description

PostCard.vue renders a single blog post as a card with a title, body text, and an edit link. Currently the component only applies `class="post-card"` to its root `<div>` with no associated styles.

Add a `<style>` section to PostCard.vue with neo-brutalist card styling: a bold solid black border, a hard offset box shadow, strong typography for the title, and a styled edit link that looks like a button.

Neo-brutalist card design:
- `.post-card`: `border: 3px solid #000`, `box-shadow: 5px 5px 0 #000`, `padding: 1.25rem`, `background: #fff`, `margin-bottom: 1.5rem`
- `.post-card:hover`: shift shadow to `3px 3px 0 #000` and translate the card by `2px 2px` to simulate press
- `.post-card h2`: `font-size: 1.4rem`, `font-weight: 900`, `text-transform: uppercase`, `letter-spacing: 0.03em`, `margin-bottom: 0.5rem`
- `.post-card p`: `margin-bottom: 1rem`, `line-height: 1.5`
- `.post-card a` (edit link): styled as a small inverted button — `display: inline-block`, `background: #000`, `color: #fff`, `padding: 0.3rem 0.75rem`, `font-weight: 700`, `text-decoration: none`, `border: 2px solid #000`; hover: `background: #fff; color: #000`

Also update the template to add appropriate CSS classes to the `<h2>`, `<p>`, and `<a>` elements so the selectors work correctly (alternatively, use element selectors scoped to `.post-card`).

## Acceptance Criteria

- PostCard.vue has a `<style>` section with all the above rules
- The `.post-card` container has a visible black border and an offset hard shadow
- The title (`<h2>`) is uppercase and heavy weight
- The edit link is styled as a black button with white text, inverting on hover
- No border-radius is used anywhere in the styles
- `go test ./examples/blog/...` passes after the change
