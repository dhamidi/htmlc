# Add neo-brutalist list styles to PostList.vue

## Description

PostList.vue renders a list of blog posts. It supports a scoped `#item` slot for customizing how each post is rendered and shows an empty-state message when there are no posts. Currently it applies `class="post-list"` to the root element and has no styles.

Add a `<style>` section to PostList.vue with neo-brutalist list styling: reset the default `<ul>` and `<li>` styles, style the empty-state paragraph to be visually impactful, and add a bottom border separator between list items.

Design:
- `.post-list ul`: `list-style: none; padding: 0; margin: 0`
- `.post-list li`: `border-bottom: 2px solid #000` for visual separation between items; remove last-child border with `.post-list li:last-child { border-bottom: none }`
- `.post-list p` (empty state): `font-size: 1.2rem`, `font-weight: 700`, `text-transform: uppercase`, `letter-spacing: 0.05em`, `padding: 2rem 0`, `border: 3px solid #000`, `text-align: center`

## Acceptance Criteria

- PostList.vue has a `<style>` section with the rules above
- The `<ul>` has no default list bullets or left padding
- Each `<li>` has a 2px solid black bottom border except the last item
- The empty-state paragraph (`<p>No posts yet.</p>`) is styled prominently with an uppercase bold look and a border box
- `go test ./examples/blog/...` passes after the change
