# Add neo-brutalist form styles to PostForm.vue

## Description

PostForm.vue renders the create/edit post form with a title input, body textarea, and submit button. It currently has no styles.

Add a `<style>` section to PostForm.vue with neo-brutalist form styling: bold uppercase labels, thick-bordered inputs and textarea with no border-radius, and a high-contrast submit button that inverts on hover.

Design:
- `form`: `display: flex; flex-direction: column; gap: 1.25rem`
- `label`: `display: block; font-weight: 900; text-transform: uppercase; letter-spacing: 0.06em; font-size: 0.85rem; margin-bottom: 0.35rem`
- `input[type="text"], textarea`: `width: 100%; border: 3px solid #000; padding: 0.6rem 0.75rem; font-size: 1rem; font-family: inherit; background: #fff; color: #000; border-radius: 0; outline: none`; focus state: `box-shadow: 4px 4px 0 #000`
- `textarea`: `min-height: 8rem; resize: vertical`
- `button[type="submit"]`: `align-self: flex-start; background: #000; color: #fff; border: 3px solid #000; padding: 0.6rem 1.5rem; font-size: 1rem; font-weight: 900; text-transform: uppercase; letter-spacing: 0.05em; cursor: pointer; box-shadow: 4px 4px 0 #000`; hover: `background: #fff; color: #000; box-shadow: 2px 2px 0 #000; transform: translate(2px, 2px)`

Also add a wrapper `<div class="form-group">` around each label+input pair in the template (they already have `<div>` wrappers, so update those to use `class="form-group"`).

## Acceptance Criteria

- PostForm.vue has a `<style>` section with all the above rules
- Labels are uppercase, bold, and small
- Text input and textarea have thick black borders with no border-radius; focus adds a hard box shadow
- Submit button is black with white text, uppercase, bold; hover inverts to white with black text
- The form fields are laid out vertically with consistent spacing
- No border-radius is used anywhere
- `go test ./examples/blog/...` passes after the change
